Using Internet standards and DNS infrastructure to support innovation in the financial ecosystem

Financial transactions are essentially about exchanging financial resources between accounts. Account numbers are financial addresses for financial transactions: they identify the source and the destination of the financial resources.

Problem Definition

  • An important percentage of payments fail because the account number is not valid.
  • Fraudsters can substitute account numbers. The payee cannot check the link between the account number and the beneficiary.
  • Sanction screening is based on names, not necessarily on account numbers

Solution

  • Using DNS to resolve Account Numbers, just like DNS is resolving URLs, to reach the account servicing bank.
  • Exposing “GetAccountDetails” at the account servicing bank
  • Controlling information access to protect privacy while providing a value add service

Value Proposition

Based on Web Standards and Web Infrastructure - Low investment - Collaborative - Innovation enabling
